# Build-a-Sorting-Visualizer-for-freecodecamp-Project
A Bubble Sort visualizer built with JavaScript that generates a random array and displays each step of the sorting process. Each comparison and swap is rendered in the DOM, with highlighted elements to illustrate how the algorithm works step by step.
This project is a Bubble Sort visualizer built using vanilla JavaScript, HTML, and CSS.

It generates a random array of numbers and visually demonstrates how the Bubble Sort algorithm works by rendering each step in the DOM. Every comparison and swap is displayed as a separate state, with the current elements highlighted for clarity.

Key features:
- Generates a random array of 5 numbers (1–100)
- Displays each step of the Bubble Sort process
- Highlights elements being compared in real time
- Shows the final sorted array with visual indication
- Ensures a clean DOM structure for each step

This project helped reinforce understanding of sorting algorithms, DOM manipulation, and step-by-step state visualization.
